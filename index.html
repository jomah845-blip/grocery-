About FreshFind

FreshFind helps Americans find authentic grocery stores — whether you're new to a city, road-tripping across the country, or looking for a specific chain in your area.

Our data is powered by Google Maps Places API, ensuring real-time accuracy for store locations, hours, and ratings. We cover all 50 states and over 50,000 grocery locations nationwide.

Setup Instructions

  1. Get a Google Maps JavaScript API key from Google Cloud Console
  2. Enable Maps JavaScript API and Places API
  3. Add your key to config.js (see repo README)
  4. Deploy to any static host or open index.html locally
